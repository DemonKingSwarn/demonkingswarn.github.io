DEBMON

DEBMON is an efficient shell script that will install a fully-featured tiling window manager-based system on any Ubuntu or Pop!_os-based system, without any of the routine of manual post-install processes and configuration.

By default, DEBMON installs and deploys my own personal dotfiles which people may know from my videos on YouTube. It will install my special build of i3wm which is very cool.

DEBMON can also be forked on Github or given command line options to customize it for your own needs: you can have it deploy your own dotfiles and install all prerequisite programs, as LARBS only needs to read a single list of programs, and the script itself is designed to be easily rearranged.

Two types of DEBMON users

Is DEBMON for you? Probably yes. I don't know how else you would've found this site. The script is for two types of people:

  1. People who already know their stuff and just want to automate installing a system without doing the boring stuff you've done a million times.
  2. Novices who want to use and learn about a leet hackerman computer setup like those in the movies for either efficiency or looking cool.

No actual phonies allowed though. The goal of the system for novices is helping you understand how a good Unix system works and how it is modified. This is not a hand-holding distro that does things automatically for you. Instead, you realize how easy it is to set things up automatically yourself.

Installation

On any (preferrably fresh) install of Ubuntu or Pop!_os, make sure git is installed and just run the following as the root user:

git clone https://github.com/demonkingswarn/debmon.git
cd debmon
chmod +x debmon.sh
sh debmon.sh

DEBMON will then guide you through installation, which is typically relatively snappy. On my slow internet, it takes around 10 minutes.

Note that the DEBMON scripts will not partition any drives or wipe anything, but when it deploys the dotfiles, it will overwrite any preexisting files: e.g. the DEBMON bashrc will replace your old bashrc, etc. To avoid even this risk, you can tell DEBMON to install for a new username and nothing will be overwritten.

  • software: i3 (window manager), bumblebee (status bar) st (terminal), dmenu, rofi, etc., all sensible builds even with the minimalist source pre-downloaded for you.
  • Minimalist and keyboard-centric programs for file management (lf), mail (neomutt), RSS (newsboat), music/audio (paplay) etc.
  • Highly extensible programs for everything else: sxiv (image viewer), zathura (pdf/djvu/epub reader).
  • A well featured and configured shell (zsh) and other core programs, including many elegant scripts.

Note that making your own fork of DEBMON is as easy as using your own dotfiles, making a programs file like the one above, and making any other direct changes to the script you want.
